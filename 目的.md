# 智能收藏夹扩展 - 项目目的与核心规格 (Purpose & Specification)

本项目旨在为浏览器用户提供一个功能强大、美观且高隐私保护的深度收藏管理工具。通过优化原生书签的使用体验，本工具实现了多级分类、标签系统以及高效的数据迁移能力。

## 🎯 核心目的 (Goals)

1.  **深度分类与组织**：提供无限层级的文件夹嵌套，解决原生书签管理器在大规模收藏下难以维护的问题。
2.  **多维度检索**：通过标签系统（Tagging）和基于 Fuse.js 的模糊搜索，让用户能从不同维度快速定位书签。
3.  **高度隐私与安全**：坚持最小权限原则（如使用 `activeTab` 替代 `tabs`），确保用户浏览数据不被滥用。
4.  **无缝数据迁移**：支持与浏览器原生书签的双向同步，并提供 JSON/HTML 的完全导出备份能力。
5.  **极致 UI/UX**：提供现代化的交互界面，支持多视图（列表/网格/卡片）切换及沉浸式的主题系统。

## 🛠️ 已实现的规格 (Specifications)

### 1. 数据架构与存储
- **本地存储**：使用 `chrome.storage.local`，并引入深度克隆（Deep Cloning）机制，确保 Vue 响应式数据在固化到硬盘时的正确性。
- **全局同步**：标签和分类采用全局注册机制，确保跨组件、跨窗口的数据一致性。

### 2. 交互与功能
- **智能导出**：支持标准 Netscape 格式的 HTML 导出。具备**空分类自动剪枝**功能，并支持导出页面的折叠交互。
- **访问追踪**：通过 Background Service Worker 实现可靠的访问量统计和最后访问时间追踪。
- **快速保存**：集成右键菜单及快捷键（`Ctrl+Shift+B`），支持在不打断用户流程的情况下快速收藏页面。

### 3. 系统优化
- **主题系统**：完美集成 Element Plus 的浅色/深色主题，支持跟随系统自动切换。
- **性能优化**：书签列表采用懒加载渲染及 debounce 搜索优化，支持数千级规模的书签流畅管理。

## 🔮 未来愿景
