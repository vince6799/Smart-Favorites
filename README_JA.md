# スマートお気に入り - 強力なブラウザブックマーク管理ツール

[English](./README_EN.md) | [简体中文](./README.md) | 日本語

スマートお気に入り (Smart Favorites) は、ブックマーク管理の効率を高めるために設計された強力なブラウザ拡張機能です。基本的なフォルダ分類だけでなく、タグシステム、全文検索、バージョン管理されたクラウドバックアップ（Supabase）、および多言語サポートを提供します。

## ✨ 機能の特徴

### 📁 コア管理
- **無限階層分類**：深いフォルダのネストをサポートし、ブックマークを柔軟に整理。
- **タグシステム**：マルチタグ対応により、カテゴリを越えたフィルタリングが可能。
- **3つの表示モード**：リスト、グリッド、カードビューを自由に切り替え。
- **スマートソート**：作成日時、訪問回数、名前、または最終訪問日時での並べ替えをサポート。
- **favicon 自動取得**：サイトのアイコンを美しく表示し、表示設定もカスタマイズ可能。

### 🔄 データ同期とバックアップ
- **Supabase クラウド同期**：バージョン管理されたストレージで、複数デバイス間の同期に対応。
- **マルチバージョン・クラウドバックアップ**：同期ごとのスナップショットを記録し、履歴（直近100件）から選択的な復元が可能。
- **自動保持ポリシー**：クラウドの保持件数をカスタマイズ可能（デフォルト20、0は無制限）。
- **ローカル自動バックアップ**：ローカルストレージへの定期的な自動バックアップ設定が可能。
- **全プラットフォーム対応メタデータ**：バックアップ元のブラウザ（Chrome/Edge/Firefox）とOS（Mac/Windows）を自動的に記録。

### ⌨️ 効率的なインタラクション
- **クイック保存**：右クリックメニューからワンクリックでページを保存。
- **グローバルショートカット**：ミリ秒単位で検索インターフェースを呼び出し、素早く検索。
- **カスタマイズ可能なショートカット**：グローバルおよびアプリ内のショートカット設定をサポート。

### 🌐 国際化
- **多言語サポート**：英語、簡体中国語、日本語を内蔵。ブラウザの言語を自動認識。

## 🚀 クイックスタート

### 開発とビルド

1. **依存関係のインストール**
   ```bash
   npm install
   ```

2. **開発モード** (ホットリロード)
   ```bash
   npm run dev
   ```

3. **拡張機能のビルド** (ブラウザ用)
   ```bash
   npm run build:extension
   ```

4. **ショーケースサイトのビルド** (GitHub Pages用)
   ```bash
   npm run build:web
   ```

### 拡張機能のインストール
1. Chrome を開き、 `chrome://extensions/` にアクセスします。
2. 「デベロッパー モード」をオンにします。
3. 「パッケージ化されていない拡張機能を読み込む」をクリックし、プロジェクト内の `dist` ディレクトリを選択します。

## ☁️ Supabase クラウド同期設定

1. [Supabase](https://supabase.com/) で新しいプロジェクトを作成します。
2. 拡張機能の「自動バックアップ」メニューで「初期化SQLをコピー」をクリックします。
3. Supabase の SQL Editor でスクリプトを実行し、必要なテーブルとセキュリティポリシー (RLS) を作成します。
4. プロジェクトの URL と Anon Key を拡張機能の設定に入力して同期を開始します。

## 🛠️ 技術スタック
- **フレームワーク**: Vue 3 (Composition API) + TypeScript
- **UI コンポーネント**: Element Plus
- **状態管理**: Pinia
- **検索エンジン**: Fuse.js (曖昧検索)
- **ビルドツール**: Vite
- **バックエンド**: Supabase (PostgreSQL / RLS)

## 📂 プロジェクト構造
```
src/
├── background/      # バックグラウンド Service Worker (右クリックメニュー、コマンド)
├── popup/           # 拡張機能ポップアップ UI (Vue コンポーネント)
├── i18n/            # 多言語対応パッケージ (EN, ZH, JA)
├── services/        # コアサービス (Storage, Supabase, Metadata)
├── stores/          # 状態管理 (ブックマーク, カテゴリ, 設定)
├── utils/           # ユーティリティ
├── manifest.json    # 拡張機能マニフェスト
├── index.html       # GitHub Pages ランディングページ (中)
├── index_en.html    # GitHub Pages ランディングページ (英)
├── index_ja.html    # GitHub Pages ランディングページ (日)
└── scripts/         # ビルドスクリプト (build-showcase.js)
```
