# 智能收藏夹 - 浏览器扩展

一款功能强大的浏览器收藏夹管理工具，支持多级分类、标签系统、搜索功能和数据导入导出。

## ✨ 功能特性

### 核心功能
- 📁 **多级分类系统** - 无限层级的文件夹嵌套，灵活组织书签
- 🏷️ **标签系统** - 多标签支持，快速筛选和分类
- 🔍 **强大搜索** - 全文搜索，支持标题、URL、描述和标签
- 📱 **现代化界面** - 简洁美观，支持深色/浅色主题
- ⚡ **快速访问** - 快捷键、右键菜单，一键保存网页

### 数据管理
- 📥 **从浏览器导入** - 一键导入浏览器原生书签
- 📤 **导出到浏览器** - 将书签导出回浏览器收藏夹
- 💾 **JSON导入导出** - 便于数据备份和迁移
- 🔄 **数据同步** - 保持原有文件夹层级结构

### 用户体验
- 🎨 **主题切换** - 浅色、深色、跟随系统
- 👁️ **多种视图** - 列表、网格、卡片视图
- 🔤 **排序选项** - 按时间、名称、访问次数排序
- 🌐 **网站图标** - 自动获取并显示favicon
- 📝 **书签备注** - 为每个书签添加详细说明

## 🚀 快速开始

### 安装依赖




## 📦 安装扩展

### 开发模式

1. 打开Chrome浏览器，访问 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择项目的 `dist` 目录

### 从Chrome 应用商店安装

 - [Chrome 应用商店](https://chromewebstore.google.com/detail/%E6%99%BA%E8%83%BD%E6%94%B6%E8%97%8F%E5%A4%B9/igpenlimhjaeobfjmljakadodcppfala)

## 🎯 使用指南

### 添加书签

**方法一：右键菜单**
- 在任意网页右键，选择"保存到我的收藏夹"
- 可选择保存到默认分类或选择目标分类

**方法二：扩展弹窗**
- 点击浏览器工具栏的扩展图标
- 点击"新建书签"按钮
- 输入URL并保存

### 管理分类

- 点击侧边栏分类旁的 `+` 按钮创建新分类
- 右键分类可编辑或删除
- 支持拖拽调整分类顺序（开发中）

### 搜索书签

- 在顶部搜索框输入关键词
- 支持搜索标题、URL、描述、标签
- 搜索结果实时更新

### 导入导出

- 点击右上角的菜单按钮
- 选择相应的导入/导出选项
- 支持浏览器书签和JSON格式

## ⌨️ 快捷键

- `Ctrl/Cmd + Shift + B` - 打开扩展弹窗
- `Ctrl/Cmd + Shift + F` - 快速搜索

## 🛠️ 技术栈

- **前端框架**: Vue 3 + TypeScript
- **UI组件库**: Element Plus
- **状态管理**: Pinia
- **构建工具**: Vite
- **搜索引擎**: Fuse.js
- **浏览器API**: Chrome Extension API

## 📂 项目结构

```
├── src/
│   ├── background/          # 后台脚本
│   │   ├── index.ts         # Service Worker

│   ├── stores/              # Pinia状态管理
│   ├── services/            # 业务逻辑
│   ├── utils/               # 工具函数
│   └── types/               # TypeScript类型
├── public/
│   ├── manifest.json        # 扩展配置
│   └── icons/               # 扩展图标
└── dist/                    # 构建输出
```
